====== Topics for Discussion ======

  * To add a new topic create a new page under the ''/topics'' heading and filling out the template
      * i.e. append ''/my-new-topic-page'' to the URL of this page, navigate there, click "Create this page", edit, and save
  * To find topics, use the tag clouds to filter and/or click table headings to sort.

==== Open Topics ====

---- datacloud ----
field: Spec_tags
min: 2
filter: %pageid% ~ topics:*
----

---- datacloud ----
field: Status_tags
filter: %pageid% ~ topics:*
----

---- datatable ----
cols    : %title%, Spec_tags, Owner_tags, Status_tags, Action
headers : Summary, Spec, Owner, Status, WG Action
filter  : %pageid% ~ topics:*
filter  : Status_tags != Closed
----
