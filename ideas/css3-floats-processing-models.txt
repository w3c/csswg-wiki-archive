===== Introduction =====

This page summarizes the processing models for the different proposals under consideration for the CSS Floats/Exclusions effort. Each is presented following to answer the following questions:

  * how is an element made an exclusion for other content? How is an element made to wrap its internal content inside a shape?
  * what is the container model for an exclusion?
  * what is affected by an exclusion that belongs to a given container?
