====== CSS Ruby ======

The [[http://www.w3.org/TR/css3-ruby/|CSS Ruby specification]] defines the CSS model for displaying ruby (inline annotations such as those typically used in Japanese documents).

===== Current Issues =====

The following are known problems in the [[http://www.w3.org/TR/2003/CR-css3-ruby-20030514/|14 May 2003 CSS Ruby Candidate Recommendation]]. We expect to publish a Working Draft that incorporates the necessary changes to resolve them.
