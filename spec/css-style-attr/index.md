# CSS Style Attributes

The [CSS Style Attributes specification](http://www.w3.org/TR/css-style-attr/) defines the processing model for a CSS style attribute in languages like HTML (which uses 'style' literally).

See the [latest editors draft of CSS Style Attributes](http://dev.w3.org/csswg/css-style-attr/)

The CSS Style Attributes spec is a CR!
