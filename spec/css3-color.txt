====== css3-color issues list ======

The [[http://www.w3.org/TR/css3-color/|css3-color specification]] defines color values and color-related properties.

===== Current Issues =====

The following are known problems in the [[http://www.w3.org/TR/2008/WD-css3-color-20080721|21 July 2008 css3-color Last Call Working Draft]].

See the [[http://www.w3.org/Style/CSS/Test/#css3-color|released]] and [[http://dev.w3.org/CSS/css3-color-test-suite/src/|development]] versions of the test suite for the "Tests" references.
