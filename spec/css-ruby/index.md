# CSS Ruby

The [CSS Ruby specification](http://www.w3.org/TR/css3-ruby/) defines the CSS model for displaying ruby (inline annotations such as those typically used in Japanese documents).

## Current Issues

The following are known problems in the [14 May 2003 CSS Ruby Candidate Recommendation](http://www.w3.org/TR/2003/CR-css3-ruby-20030514/). We expect to publish a Working Draft that incorporates the necessary changes to resolve them.
