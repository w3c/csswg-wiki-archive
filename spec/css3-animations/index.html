====== CSS3 Animations ====== The [[http://www.w3.org/TR/css3-animations/|CSS Animations Module Level 3]] (CSS3-Animations, latest public TR version) defines visual animations using CSS. For latest edits/resolutions, see the editors draft: * http://dev.w3.org/csswg/css3-animations/ For unresolved issues, see Bugzilla: * https://www.w3.org/Bugs/Public/buglist.cgi?query_format=advanced&product=CSS&component=Animations&bug_status=NEW&bug_status=ASSIGNED&bug_status=REOPENED To raise an issue with the Animations spec, please open a bug [[https://www.w3.org/Bugs/Public/buglist.cgi?product=CSS&component=Animations&resolution=---|in Bugzilla]]. ====Bug Triage==== Bugs are current categorized as * [[https://www.w3.org/Bugs/Public/buglist.cgi?quicksearch=component%3Aanimations+whiteboard%3A%5Beasy%5D | Easy]] - have a clear proposed solution * [[https://www.w3.org/Bugs/Public/buglist.cgi?quicksearch=component%3Aanimations+whiteboard%3A%5Bmedium%5D| Medium]] - have a good chance of being settled after a brief discussion * [[https://www.w3.org/Bugs/Public/buglist.cgi?quicksearch=component%3Aanimations+whiteboard%3A%5Bhard%5D | Hard]] - require more mailing discussion and a proposal * [[https://www.w3.org/Bugs/Public/buglist.cgi?quicksearch=component%3Aanimations+whiteboard%3A%5Bpostpone%5D| Postpone]] - move to Level 4
