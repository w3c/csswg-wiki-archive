CSS3 Animations

The CSS Animations Module Level 3 (CSS3-Animations, latest public TR version) defines visual animations using CSS.

For latest edits/resolutions, see the editors draft:

  • http://dev.w3.org/csswg/css3-animations/

For unresolved issues, see Bugzilla:

  • https://www.w3.org/Bugs/Public/buglist.cgi?query_format=advanced&product=CSS&component=Animations&bug_status=NEW&bug_status=ASSIGNED&bug_status=REOPENED

To raise an issue with the Animations spec, please open a bug in Bugzilla.

Bug Triage

Bugs are current categorized as

  • Easy - have a clear proposed solution
  • Medium - have a good chance of being settled after a brief discussion
  • Hard - require more mailing discussion and a proposal
  • Postpone - move to Level 4
