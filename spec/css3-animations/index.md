# CSS3 Animations

The [CSS Animations Module Level 3](http://www.w3.org/TR/css3-animations/) (CSS3-Animations, latest public TR version) defines visual animations using CSS.

For latest edits/resolutions, see the editors draft:

- <http://dev.w3.org/csswg/css3-animations/>

For unresolved issues, see Bugzilla:

- <https://www.w3.org/Bugs/Public/buglist.cgi?query_format=advanced&product=CSS&component=Animations&bug_status=NEW&bug_status=ASSIGNED&bug_status=REOPENED>

To raise an issue with the Animations spec, please open a bug [in Bugzilla](https://www.w3.org/Bugs/Public/buglist.cgi?product=CSS&component=Animations&resolution=---).

### Bug Triage

Bugs are current categorized as

- [Easy](https://www.w3.org/Bugs/Public/buglist.cgi?quicksearch=component%3Aanimations+whiteboard%3A%5Beasy%5D) - have a clear proposed solution
- [Medium](https://www.w3.org/Bugs/Public/buglist.cgi?quicksearch=component%3Aanimations+whiteboard%3A%5Bmedium%5D) - have a good chance of being settled after a brief discussion
- [Hard](https://www.w3.org/Bugs/Public/buglist.cgi?quicksearch=component%3Aanimations+whiteboard%3A%5Bhard%5D) - require more mailing discussion and a proposal
- [Postpone](https://www.w3.org/Bugs/Public/buglist.cgi?quicksearch=component%3Aanimations+whiteboard%3A%5Bpostpone%5D) - move to Level 4
