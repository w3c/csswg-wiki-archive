====== CSS Style Attributes ======
The [[http://www.w3.org/TR/css-style-attr/|CSS Style Attributes specification]] defines the processing model for a CSS style attribute in languages like HTML (which uses 'style' literally).

See the [[http://dev.w3.org/csswg/css-style-attr/|latest editors draft of CSS Style Attributes]]

The CSS Style Attributes spec is a CR!
