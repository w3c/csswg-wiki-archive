# Paris F2F August 2015

- Host: Mozilla
- Dinner 7pm, Le Zinc des Cavistes
- Dates: Tuesday-Thursday, August 25-27, 2015
- Nearby events:
  - [Editing TF meeting](https://lists.w3.org/Archives/Public/public-editing-tf/2015Jun/0023.html) (Sunday-Monday 23-24)
  - [Houdini meeting](https://wiki.css-houdini.org/planning/paris-2015) (Friday-Saturday 28-29)

### Meeting Minutes

##### Tuesday, 25 August

- [Part I](https://lists.w3.org/Archives/Public/www-style/2015Sep/0037.html): CSS Cascade, CSS Fragmentation, CSS Animations, CSS Text, TPAC Japanese Industry Meet-up, Number of F2F Meetings Per Year
- [Part II](https://lists.w3.org/Archives/Public/www-style/2015Sep/0038.html): Flexbox, Grid/Flex Percentages, Testing
- [Part III](https://lists.w3.org/Archives/Public/www-style/2015Sep/0040.html): CSS UI 4, CSS Text 4, User Stylesheets
- [Part IV](https://lists.w3.org/Archives/Public/www-style/2015Sep/0041.html): Snapshot 2015 and Prefix Policy, @apply rule

##### Wednesday, 26 August

- [Part I](https://lists.w3.org/Archives/Public/www-style/2015Sep/0093.html): Round Display, Grid Layout
- [Part II](https://lists.w3.org/Archives/Public/www-style/2015Sep/0094.html): Animations, Scroll Snapping
- [Part III](https://lists.w3.org/Archives/Public/www-style/2015Sep/0095.html): Defining Pagination, CSS Priorities from DigiPub
- [Part IV](https://lists.w3.org/Archives/Public/www-style/2015Sep/0095.html): CSS Inline

##### Thursday, 27 August

- [Part I](https://lists.w3.org/Archives/Public/www-style/2015Sep/0109.html): FXTF Meeting Part 1- SVG Resources, SVG Images Without Intrinsic Size, Matrix Interpolation Revisited, SLERPing and 0deg angles
- [Part II](https://lists.w3.org/Archives/Public/www-style/2015Sep/0110.html): FXTF Meeting Part 2- glyph-orientation, writing-mode Values from SVG 1.1, getTransformToElement, Path Animation, Zoom Features for Media Queries
- [Part III](https://lists.w3.org/Archives/Public/www-style/2015Sep/0111.html): Selectors
- [Part IV](https://lists.w3.org/Archives/Public/www-style/2015Sep/0112.html): will-change, Scroll Snap, Input Modality
- [Part V](https://lists.w3.org/Archives/Public/www-style/2015Sep/0113.html): Page Floats, Writing Modes, Snapshot 2015, Flexbox % Follow-Up
