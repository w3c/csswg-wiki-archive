==== Proposed Agenda ====

  * Environment Variables spec --???
  * Initial Letter Layout Model and other CSS Inline Layout Issues (could this happen Monday? --dauwhe)
  * [[https://wicg.github.io/construct-stylesheets/index.html|Constructable Stylesheets]] (Tuesday) -- Rakina & Eric & Tab
    * [[https://github.com/WICG/construct-stylesheets/issues/23|Can the same constructed stylesheet be used in two different Documents?]]
    * [[https://github.com/WICG/construct-stylesheets/issues/15|Need to define what fetch groups various loads from the sheet go in]]
    * [[https://github.com/WICG/construct-stylesheets/issues/34|Can we use CSSStyleSheets from <style>/<link> in adoptedStyleSheets?]]
    * [[https://github.com/WICG/construct-stylesheets/issues/36|Should we use FrozenArray instead of StyleSheetList for adoptedStyleSheets]]
    * Should this spec move from WICG to CSS WG?
    * Blink is interested in shipping. 
  * [[https://github.com/w3c/csswg-drafts/issues/3118|text-underline-offset]]
  * [[https://drafts.csswg.org/css-text-4/#text-spacing-property|text-spacing]]
  * [[https://github.com/w3c/csswg-drafts/issues/1744|Propose adding lang as a font-face descriptor]] ideally with I18n Core 
  * Identify new people to represent the CSSWG in the CSS a11y task force
  * Scrollbar Styling - FPWD, experimental implementation(s), and maybe even demo(s)?
  * CSS-UI
    * Path forward for the appearance property (https://github.com/w3c/csswg-drafts/issues/3024)—Florian
    * fieldset/legend (https://github.com/w3c/csswg-drafts/issues/3094)—Florian
    * Dark mode and native form controls (discussion)—fremy
    * Should we define outline more tightly and aim for close interop, or leave it open to UI innovation? (see [[https://github.com/w3c/csswg-drafts/issues/3184|3184]] for a recent example, but also the spec is full of "may")—Florian

  * Apple's color-filter proposal (status gathering)—fremy
  * Spatial Navigation—Jihye&Florian (prefers Tuesday, not during AC meeting, definitely avoid Monday early afternoon and avoid from 5pm Tuesday)
  * Effect of filter element on root element https://github.com/w3c/fxtf-drafts/issues/282 --chrishtr
  * backdrop-filter spec issues: https://github.com/w3c/fxtf-drafts/issues/53
  * [css-fonts-4] src: local() font unique name matching ambiguous & restricts matched locale https://github.com/w3c/csswg-drafts/issues/3177
  * [css-overscroll-behavior] Moving [[https://drafts.csswg.org/css-overscroll-behavior/|spec]] to FPWD. This was recently migrated from WICG.
  * [[https://wicg.github.io/scroll-animations/ | Scroll-linked Animations ]] - Spec is on WICG and Blink now has a partial experimental implementation (majidvp@ can demo). Gathering status and see if there is interest in moving it (or a subset) forward.
  * Accessibility API Mappings (https://w3c.github.io/html-aam/) - What it is, how it matters to CSS, when we need to ping a11y people about implications. Focus on display:contents being unexpectedly troublesome. -- Alice Boxhall
  * Css-text-3:
    * Revisit text-align shorthanding text-align-last: https://github.com/w3c/csswg-draft/issues/3117
    * word-wrap/overflow-wrap: break-word should affect min-content https://github.com/w3c/csswg-drafts/issues/2682
