====== Oslo F2F August 2010 ======

Topics for the August 2010 CSS Working Group Face-to-Face meeting in Oslo, Norway.

  * Review Vendor Prefix policy - what happens when CR reverts to WD, etc?
  * Define ::selection 
  * CSS3 Text Layout - discuss vertical text support, physical vs. logical properties, impact on HTML, other CSS3 modules (1 hour)
  * Font feature support in CSS3 Fonts (2 hours)
  * CSS 2.1 Test Suite - current status, remaining issues (1 hour)
  * Charter update (1 hour)
  * CSS 2.1 Issues and Status (1 day)
  * box-shadow blur definition
  * <meta name=viewport> - add something equivalent to CSS and define the then <meta> equivalent as deprecated?
  * Discuss Media Queries Test Suite. http://lists.w3.org/Archives/Public/www-style/2010Jul/0448.html 
  * [[ideas:pointer-events|Hit testing]]
  * GCPM, esp. hyphenation properties http://www.w3.org/TR/css3-gcpm/#hyphenation
