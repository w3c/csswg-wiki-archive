Meeting Planning Pages

This is the CSS Working Group's planning space.

The chairs will add most events to the W3C calendar, which should show up in your W3C calendar (see export options at the bottom) if you're a member of the CSS Working Group. An unofficial CSSWG Meeting iCalendar Feed is also available.

All meetings use IRC for live minuting and as a backchannel: the #css channel on the W3C IRC server.
