## About DokuWiki

DokuWiki was chosen because

- it has decent [markup](../wiki/syntax/ "wiki:syntax") capabilities with relatively clean HTML output
- it can accept HTML markup
- it can support an advanced structured text language ([Markdown](http://daringfireball.net/projects/markdown/)) with sensible list, block, and embedded HTML behavior.
- it has hierarchical structure (with breadcrumbs!)
- it has clean URLs
- it has a a simple, uncluttered interface
- it has automatic yet unobtrusive \<html\>\<abbr title=“Table of Contents”\>ToC\</abbr\>s\</html\>
- it has flexible access control
- it stores data in human-editable text files
- it has revision control
- it's customizable
- it's open source

It is [fantasai](http://fantasai.inkedblade.net/)'s dream wiki~
