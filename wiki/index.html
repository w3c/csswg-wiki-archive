====== About This Wiki ====== This wiki is a [[https://www.dokuwiki.org/DokuWiki|DokuWiki]]. * [Syntax](/wiki/syntax) * [Index](/?do=index) * [Recent Changes](/?do=recent) ===== About DokuWiki ===== DokuWiki was chosen because * it has decent [[wiki:syntax|markup]] capabilities with relatively clean HTML output * it can accept HTML markup * it can support an advanced structured text language ([[http://daringfireball.net/projects/markdown/|Markdown]]) with sensible list, block, and embedded HTML behavior. * it has hierarchical structure (with breadcrumbs!) * it has clean URLs * it has a a simple, uncluttered interface * it has automatic yet unobtrusive ToCs * it has flexible access control * it stores data in human-editable text files * it has revision control * it's customizable * it's open source It is [[http://fantasai.inkedblade.net/|fantasai]]'s dream wiki~
