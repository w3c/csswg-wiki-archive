====== Installing Fonts for CSS Testing ======

**To install the Ahem font on Windows:**
<html>
<head><title="" /></head>
<body>
    <ul>
        <li>Download the font you want to install. The Ahem font can be obtained at <a href="http://dev.w3.org/CSS/fonts/ahem/">http://dev.w3.org/CSS/fonts/ahem/</a>.</li>
        <li>Open the folder where you downloaded the font file.</li>
        <li><strong>For Windows XP/Server 2003:</strong>
            <ul>
                <li>From the run dialog type '%windir%/fonts' and press <em>enter</em>.</li>
                <li>From the download folder, where you downloaded the font; drag-and-drop the file to the fonts folder.</li>
            </ul>
        </li>
        <li><strong>For Windows Vista/Server 2008</strong>
            <ul>
                <li>From the download folder, where you downloaded the font; right-click the font and select <em>install</em>.</li>
                <li>Confirm your acceptance to the User Account Control (UAC) pop-up window. Select <em>continue</em></li>
            </ul>
        </li>
    </ul>
</body>
</html>
