    ===== Rendering page boxes that do not fit a page sheet =====

  49. XXX If a page box does not match the target page sheet dimensions, the user agent must either render the page box at the indicated size on a larger page sheet; or rotate the page box 90° if this will make the page box fit the page sheet; or scale the page box to fit the page sheet; or reformat the page contents, including 'spilling' onto other page sheets; or clip overflowed content. :

    ===== Cascading in the page context =====

  55. The specificity of an @page rule is computed by concatenating the numbers f-g-h where f=1 if an @page rule has a named page; g=1 if an @page rule has a ':first' pseudo-class; and h=1 if an @page rule has a ':left' or ':right' pseudo-class. :
    The specificity of @page rules is computed in an analogous manner to the computations defined in the Selectors module:
    • if an @page rule has a named page, f=1
    • if an @page rule has a ':first' pseudo-class, g=1
    • if an @page rule has a ':left' or ':right' pseudo-class, h=1
    Concatenating the three numbers f-g-h gives the specificity.

("foo[n]" stands for the used value of property foo of the n'th box and "x²" for the square of x.)

    'top left', 'left top': Same as '0% 0%'. 'top', 'top center', 'center top': Same as '50% 0%'. 'right top', 'top right': Same as '100% 0%'.
  240. The 'left' value of 'image-position' is the same as 0% for the horizontal alignment value. :
    'top left', 'left top': Same as '0% 0%'. 'left', 'left center', 'center left': Same as '0% 50%'. 'bottom left', 'left bottom': Same as '0% 100%'.
  241. The 'center' value of 'image-position' is the same as 50% for either the horizontal alignment value (if it is not otherwise given) or the vertical alignment value (if it is not otherwise given). :
    'top', 'top center', 'center top': Same as '50% 0%'. 'left', 'left center', 'center left': Same as '0% 50%'. 'center', 'center center': Same as '50% 50%'. 'center', 'center center': Same as '50% 50%'. 'bottom', 'bottom center', 'center bottom': Same as '50% 100%'.
