## Random Notes

- Ms2ger [mentioned](http://lists.w3.org/Archives/Public/public-css-testsuite/2012Aug/0049) that CSSOM tests submitted last time often disagreed with the specification. Reviewers targeting this spec should probably spend some time getting familiar with [WebIDL](http://dev.w3.org/2006/webapi/WebIDL/) (and perhaps some concepts in ECMAScript5). If anyone can help with [the translation effort](http://www.w3.org/html/ig/zh/wiki/WebIDL), it will be much appreciated!
- Reviewers targeting css3-flexbox might want to spend some time on [Chapter 10 of CSS 2.1](http://www.w3.org/TR/CSS2/visudet.html) (Visual formatting model details) too ([translation available](http://www.w3.org/html/ig/zh/wiki/CSS2/visudet/zh-hans)).
- For presenting “File the best bug”, Zhiqiang is welcome to use any parts from fantasai's [slides](http://fantasai.inkedblade.net/style/talks/filing-good-bugs/) (or so I think. I am certainly fine with my translation reused. I am almost sure fantasai's fine but you might want to check with her). It's language negotiated (to /en, /zh-hant or /zh-hans).
